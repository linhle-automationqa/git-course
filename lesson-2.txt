1.
- git init: khởi tạo thư mục được quản lí bởi Git
- git add <file_name>: thêm file <file name> vào khu vực chờ commit (vùng staging)
- git add .: thêm tất cả các file vào khu vực chờ commit (vùng staging)
- git commit -m "message":  Tạo ra 1 phiên bản dựa vào các file đã được thêm ở vùng staging vào Repository với thông điệp là "message"
